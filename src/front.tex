\begin{abstract}
This document is an introductory course to operating system
programming with emphasis on process communication. The main novelty
of this work is the use of the Objective Caml language, a language
from the ML family, instead of the C language which is usually
associated with system programming. This peculiarity results in new
insights into both system programming and the ML language.
\end{abstract}
